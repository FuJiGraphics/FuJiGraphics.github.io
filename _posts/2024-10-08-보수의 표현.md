컴퓨터는 실수를 표현하는 다양한 방법 중에서 주로 2의 보수를 사용하여 표현한다.  
내가 생각하기에는 2의 보수를 사용하는 이유는 실수 판별을 위한 비트를 하나만 사용할 수 있기 때문이 아닐까 싶다.  
예를 들어, 정수 2에 대한 비트에 2의 보수를 취하면 다음 처럼 된다.  

'''C++
2  // 0010   (실수의 양수 2)
~  // 1101   (1의 보수)
+1 // 1110   (2의 보수)
'''

여기서 실수에서의 양수 2는 2의 보수를 취한 1110이 메모리에 적재된다.
음수는 반대로 1110에 2의 보수를 취한 0010이 메모리에 적재된다.

'''C++
-2  // 1110   (실수의 음수 2)
~   // 0001   (1의 보수)
+1  // 0010   (2의 보수)
'''
