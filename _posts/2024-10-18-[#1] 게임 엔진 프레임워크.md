### 충돌 처리

나무를 패면 나뭇가지가 내려오고 나뭇가지를 피해서 나무를 오랫동안 패면 생존하는 게임이다.

나뭇가지에 플레이어가 맞게 되면 게임은 패배하게 된다.

해당 게임을 만들기 위해서는 프레임워크도 유니티나 언리얼 엔진에서의 충돌 시스템처럼 충돌 체크를 도와주는 기능을 구현해야 한다.

OnCollide에 충돌 대상에 대한 정보를 출력해주면 아래 이미지처럼 나뭇가지와 충돌하게 되면 콘솔 프롬포트에 관련 내용을 출력해주는 것을 확인할 수 있다.

### 콜라이더 매니저 클래스

```c++
#pragma once  
#include "Core.h"  
#include "Collider.h"  
  
namespace fz {  
	  
	class ColliderManager  
	{  
		using iter = std::vector<Collider**>::iterator;  
		using const_iter = std::vector<Collider**>::const_iterator;  
  
	public:  
		void	Attach(Collider** collider);  
		void	Detach(Collider** collider);  
		iter	Find(Collider** target);  
  
		iter			begin();  
		iter			end();  
		const_iter		begin() const;  
		const_iter		end() const;  
		  
		static ColliderManager&	GetInstance();  
  
	protected:  
		explicit ColliderManager();  
		virtual ~ColliderManager();  
  
	private:  
		std::vector<Collider**>		m_ColArray;  
		static ColliderManager*		s_instance;  
	};  
  
} // namespace fz  
```

이 클래스를 통해서 시스템에 올라와있는 객체들(Layer)들 중에 콜라이더 기능을 On한 객체들을 대상으로 충돌 검출을 한 후 OnCollide를 호출해주고 있다.

### 이후 개선 사항

- 객체들을 선택해서 충돌 검출을 선택적으로 허용하도록 하는 기능
- 현재는 직사각형의 AABB 알고리즘을 통해 충돌 검출을 하고 있지만, 원 또는 다각형을 통해 충돌 검출을 할 수 있도록 하기
- 현재 메모리 할당과 별개로 프로그램이 끝나면 의문에 메모리릭이 발생함. 원인을 파악
